This project leverages cutting-edge AI tools to streamline the asset pipeline and design architecture:

Meshy AI (3D Assets):
    Used for generating 3D molecule and atom models.
    Used the PBR texturing process to ensure consistent visual quality.

Google Gemini (Design & Logic):
    Designed the core Bonding Logic and reaction algorithms.
    Optimized complex C# architectural patterns for the "Reaction Plate" mechanism.

Luma Labs (UI/UX Design):
    Used for conceptualizing the Discovery Library layout.
    Designed the visual style for the Molecule Inspector and the laboratory interface.
